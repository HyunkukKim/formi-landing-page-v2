## 🚀 시작하기

1. 저장소를 클론합니다:
```bash
git clone https://github.com/HyunkukKim/formi-landing-page-v2.git
```

2. `index.html` 파일을 웹 브라우저에서 열어 확인합니다.

3. 또는 GitHub Pages를 활성화하여 웹사이트로 접근할 수 있습니다.
